# Authgear Authflow Visualizer

A visual node-based editor for building and visualizing [Authgear Authentication Flows](https://github.com/authgear/authgear-server/blob/main/docs/specs/authentication-flow.md).

## Tech Stack
- **React + Vite**
- **React Flow** (Canvas engine)
- **Tailwind CSS + Shadcn UI** (Styling)
- **js-yaml** (YAML processing)

## Setup
```bash
npm install
npm run dev
```

Built for the Vibe Code Session.
